# 梨花祭2026特設サイト ユーザーガイド

## サイト概要

**目的**: 梨花祭2026の来場者と生徒に、企画情報、アクセス方法、イベント詳細を提供する情報ハブ

**アクセス**: 公開後、提供されるURLからアクセス可能

**対象**: 一般来場者、保護者、在校生、教職員

## Powered by Manus

このサイトは最新のWeb技術で構築されています。

**フロントエンド**: React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui による高速で美しいユーザーインターフェース

**デプロイメント**: Vercelによる自動スケーリングインフラとグローバルCDNで、世界中どこからでも高速アクセス

**レスポンシブデザイン**: モバイル、タブレット、デスクトップすべてのデバイスで最適な表示

## サイトの使い方

### トップページ

サイトにアクセスすると、梨花祭2026のテーマ「未来へ繋ぐ、私たちの物語」が表示されます。開催日時は「2026年7月19日（土）10:00〜16:00」です。「企画を見る」をクリックすると全企画一覧へ、「アクセス」をクリックすると会場案内へ移動します。

### 企画を探す

「企画紹介」ページでは、クラス発表、部活動発表、飲食企画を検索できます。画面上部の「すべて」「クラス発表」「部活動」「飲食」ボタンでカテゴリを絞り込み、検索ボックスでキーワード検索が可能です。各企画カードの「詳細を見る」をクリックすると、開催場所や時間、メニュー情報が確認できます。

### アクセス情報を確認

「アクセス・会場案内」ページで、梨花学園への行き方とスクールバスの時刻表を確認できます。「校内マップを見る」をクリックすると、各企画の場所や施設の位置が分かるマップが表示されます。

### イベント情報をチェック

「イベント情報」ページで、中夜祭（7月18日16:30〜）、後夜祭（7月19日16:30〜）、梨花祭賞の投票方法を確認できます。「投票ページへ」をクリックすると、お気に入りの企画に投票できます。

### リアルタイム情報

「リアルタイム情報」ページでは、当日の売り切れ情報、落とし物情報、最新のお知らせを確認できます。公式Instagramへのリンクもあり、ハッシュタグ「#梨花祭2026」で最新情報をチェックできます。

## サイトを管理する

### 管理UIへのアクセス

画面右上のアイコンをクリックすると、管理UIパネルが開きます。

**プレビュー**: 開発サーバーのライブプレビューを表示。変更がリアルタイムで反映されます。

**コード**: ファイルツリーでプロジェクト内のすべてのファイルを閲覧・編集できます。「すべてのファイルをダウンロード」で一括ダウンロードも可能です。

**ダッシュボード**: サイトの公開状態、アクセス統計（UV/PV）を確認できます。

**設定**: 複数のサブパネルがあります。
- 一般設定でサイト名とロゴを変更
- ドメイン設定で独自ドメインを設定
- 通知設定で通知機能を管理
- シークレット設定で環境変数を安全に管理

### チェックポイントの保存

重要な変更を行った後は、チェックポイントを保存してください。Manusに「チェックポイントを保存して」と伝えると、現在の状態が保存されます。保存後、管理UIヘッダーの「公開」ボタンが有効になります。

### サイトの公開

チェックポイント保存後、管理UIヘッダー右上の「公開」ボタンをクリックすると、サイトが本番環境にデプロイされます。数分で世界中からアクセス可能になります。
